## What is it ?

[Github Timeline](https://semanteo.is-a.dev/Github-Timeline/) is a website to see in a timeline the repos of **who ever you want !**

## How it works ?

1. Enter the username you want
2. Choose between the ascending or descending order
3. Click on **Generate !**

And it's finish ^^
